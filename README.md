# Almanaque_1932_1937
Este repositório conterá o banco de dados que deu origem ao Almanaque de Dados Eleitorais: Primeira Fase da Justiça Eleitoral (1932-1937), a ser publicado em breve pela editora do Tribunal Superior Eleitoral (TSE) e que ficará disponível para download gratuito no catálogo de publicações da mesma instituição.
